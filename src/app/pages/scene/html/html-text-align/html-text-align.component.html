Aligning Text

A useful heading attribute is align. We are dealing with three options in align: left align, center align and right align. Depending on the tag that we give, the text will be aligned left, center or right.

Inserting Images

Many a times in an HTML document, the requirement for inserting an image arises. Images are inserted using a <img> tag. The <img> tag embeds an image in an HTML page and src specifies the source / URL of an image. The height and width of the image can also be entered.
