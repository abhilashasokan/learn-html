Introductory Tags- Head, Title, Body Tags

HTML code is generally typed into notepad. Open notepad and type the following. Each one of those is a tag. There is a starting tag and a closing tag. To make a closing tag just add a / to the starting tag. Most, but not all tags have a closing tag. Think of tags as enclosing a bit of text for the browser to interpret. The browser will interpret everything between <html> and </html> as an HTML document. Different tags are interpreted different ways by the browser. Please note that it's important that you nest tags properly as explained earlier. Once you type in an opening or starting tag please type in the closing tag also. You should not wait till the end of the document to insert the corresponding closing tags. Every HTML document needs a pair of head tags

First, a <tag> tells the browser to do something. An attribute goes inside the <tag> and tells the browser how to do it.

Second point is about defaults. As you probably know, the default value is a value that the browser assumes if you have not told it otherwise. A good example is the font size. The default font size is 3 (usually). Take note that quotation marks are being used. Whenever you have an attribute="value", the value should be in quotes.
