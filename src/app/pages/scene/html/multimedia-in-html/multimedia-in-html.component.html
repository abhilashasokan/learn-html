Adding Multimedia files in HTML

Adding audio and video files enhances the appearance of the website. In this section we are going to focus on adding audio and video files. For the sake of convenience only the easiest methods are discussed.

Audio files

Kindly make sure that you have an audio file for this exercise. Audio files can be in many formats like .wav, v.cda, . mp3 .flac, .ogg etc. Depending on the format the tag should also be modified. The tag used in back-ground sound source.

You can use the tag <bgsoundsrc="1.mp3" loop="3"> if you want input the number of times the track should be played. In this example it is played 3 times.

Adding video files

Using dynscr attribute

You can add video files you the dynamic source dynscr attribute. The video file has been named as video and it is in mpg format (formats can differ). Alternatively we could use <embed src="1.mp3"> and <embed src="video.mpg"> for adding the corresponding audio and video files. Some browsers may not support certain file types in audio and video.

Forms

A form is used to collect information from users. An HTML form provides for data gathering functionality which greatly enhances its utility.

The basic construction of a HTML form is as follows:

<form> - begin a form
<input> - type of input desired whether its text or password etc
</form> - end a form
Initiate the form.

The most common type of form <input> is text.

Every input needs a name. When the user types in an address (for example, 123 baker junction), it will become the input's value and be paired with the input's name (address) and the end result after processing will be the name=value pair.

We can, if we want, type in a value. This will automatically pair the value '123 Baker junction' with the name address, unless the user changes it. Important be sure to use quotes.
We can specify the size of the text input box. As you can see, the default value is (usually) 20. You probably already know, by the way, that the default value is the value that the browser assumes if you have not told it otherwise.
