Four Key Concepts of HTML

  1. Elements

    All HTML pages are made up of elements. An element is just like a container in which a portion of a page is placed. An element consists of two tags namely, an opening tag and a closing tag. Whatever is contained inside the element will take on the characteristics of the element. For example, to identify a heading on a page, the heading matter should be enclosed in a heading element as <h1></hl>. The elements which do not require a closing tag are known as empty elements. In the case of empty elements a closing slash is added after the element name. A space is also added between the element name and the slash like. For example, For a line break, the tag will be - <br />

  2. Tags

    Tags are commands written in text document (in Note Pad). They are not displayed in the Web pages. A Tag begins a "less than" sign (<), then the element name, followed by a â€œgreater than" (>) sign. Such a tag is called as an opening tag. In the case of a closing tag, it includes a "back slash" (/) before the element name.
    For example:
    Opening tag for the paragraph element will be - <p>
    Closing tag for the paragraph element will be - </p>

  3. Attributes and Values

    Attributes are another important part of HTML. An attribute is used to define the characteristics of an element and is placed inside the element's opening tag. For example;
    To specify the size of an image or graphics on the Web page, the image element is to be used with height and width as
    <img height="" width="" />
    Here, height and width are attributes. The attributes should be followed by their values. For example, if the size of the image is 300 x 200 pixels, the HTML statement will be
    <img height= "300" width="200" />

  4. Nesting

    Nesting means arrangement of the elements in such a way that there is no overlapping of elements. Properly nested elements are contained inside one another, as in the following.
    <a><b><c></c></b></a>
