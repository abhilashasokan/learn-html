Frames

This lesson is about dividing a page into frames. The basic concept is that each frame is a regular, complete HTML document. If you wanted to divide your page into 2 side by side frames, then you would put one complete HTML document in the left frame and another complete html document in the right frame. In addition you need to write a third html document. This MASTER PAGE (this is explained in the following pages) contains the <frame> tags that specify what goes where. There are only two main frame tags to contend with: <frameset> and <frame>. Let's give each html file a name. Create two more HTML files.

The <frameset> tag does all the dividing. It divides up windows. It specifies a few things regarding how to divide them up, but remember whenever you want to do some dividing, use <frameset>.
