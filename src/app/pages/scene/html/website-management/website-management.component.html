Website management

Website management refers to the administration and control of the hardware and software used in a Web site. There are a variety of monitoring and analysis tools that report on usage, missing links and files as well as performance, all of which keeps a site intact and running smoothly.

The following steps form the crux of website

  1. Choosing a Web Hosting Service

    There are many Web hosting companies on the internet, Depending on their requirements the users are able to select the most appropriate one. It depends on;

    1. Type of web space that the user requires. E.g., free web space, shared hosting, dedicated server etc.
    2. If it is an e-commerce site, the user will need more security.
    3. Disk space, email service are additional features.
  2. Launching your website

    Usually, when you join a shared Web hosting plan, the hosting company gives you a number of facilities, such as a Web-based control panel for managing your site, some usage statistics, and email accounts. Nearly all Web hosting plans include at least the following:

    1. Web space: This is an area of the Web server's hard drive that stores your websife's files, such as Web pages and images.
    2. FTP access: This allows you to use a File Transfer Protocol program to upload your website's files to the Web space.
  3. Setting up a domain name

    A domain name is a unique name that identifies a website or computer on the Internet. Find a domain name that best suites you and get it registered with the registrar of domain names.

  4. Uploading your website
    1. Find index.html borne page file on your hard disk.
    2. Open the FTP program and use it to connect to the site
    3. Upload your home page.
    4. Test the page. Open up your Web browser and go to the URL of your website.
  5. Maintenance

    Once your website is functional you may have to focus on the following;

    1. Moving a website to a new server
    2. Choosing an email news letter system
    3. Transfer domain names between registrars
