The Analyze Menu

The Analyze Menu of SPSS Software is the most important and most commonly used Menu. Most of the statistical analysis are in this Menu. Details of this Menu will be discussed as the following topics.

Four of the Descriptive Statistics sub menu will be discussed below.
  1. Frequencies
  2. Explore
  3. Crosstabs
  4. Descriptives

1. Frequencies

The Frequencies procedure produces summary measures for categorical variables. It can be prepared in the form of frequency tables, bar charts, or pie charts. A frequency table displays counts and percentages for each distinct value found in a variable.
  1. Analyze > Descriptive frequencies > Frequencies
  2. Move the variable(s) that the user wants to prepare frequencies for into the Variable(s): box. In the example from DataSet1 we have transferred Gender and 12th exam board into the Variable(s): box
  3. Click 0K
The output will be displayed as follows in the Output Viewer.

The frequencies table given by SPSS will have columns of the variables, frequency, its percentage, valid percentage and cumulative percentage. If your data has missing values, the percentage columns will not give the actual percentages of the variable under consideration. Valid percentages calculates the percentages based on the actual percentages excluding the missing values. Cumulative percentage column give the cumulative value of each classes.

2. Explore

Explore is another basic option used in SPSS. Explore helps the user get an idea about the variable that is under study. DataSet 1 is used in this example.

  1. Analyze > Descriptive Statistics > Explore
  2. Move any of the variables into the Dependent List box. To illustrate, age of the respondents is chosen.
  3. In the Display options radio button, select Statistics
  4. Click 0K

The output is given as follows in the Output Viewer. Case Processing Summary gives the details of the variables, number of cases in the variable, if there are any missing values etc. Descriptives gives a very detailed output, mean, median, variance, standard deviation, minimum, maximum, range interquartile range, skewness and kurtosis of the variable.
