Data Processing

Any manipulation of data such as arithmetic operations, classifying, sorting etc. is called data processing. Data processing is, "the collection and manipulation of items of data to produce meaningful information." The data processing involves various processes like validation, sorting, summarizing, aggregation, analysis and reporting.

The term data processing is typically used for the initial stage of data handling followed by a data analysis in the second stage of the overall data handling.

Data Processing: Methods & Types

As mentioned above, Data processing refers to methods that take the raw data and convert it into usable information to effectively take decisions. In earlier days, data processing was done manually, and now it is done by computers. Following are some of the methods of Data Processing

Methods of Data Processing

  1. Batch Processing:

    Batch processing is the basic form of data processing. It is useful when the business enterprise has a large volume of data which require only basic processing. A branch of a retail shop for example, can batch-process all the transactions at the end of each day and send the results to the head office.

  2. Real Time Processing

    Real time Processing is done exactly at the time, when the transaction (process) is done. There are certain cases when batch processing is not useful. E.g., when a customer withdraws money from an ATM, the processing of data has to be done real time. Where batch processing is done done for large volume of data at predefined timings, real-time processing is done for smaller volume of data and is done continuously.

  3. Data Mining

    Data mining refers to the process of discovering or identifying patterns in large data from multiple sources. Generally, sources from multiple sources are combined, analyzed and checked for correlations. e.g, department store may analyze data on customers' purchases and discover, that customers who buy rice often buy bananas with it. The department store can use this information to increase sales, by placing bananas near to rice. They can also discover which product sells more and which less etc.

  4. Statistical Processing

    Statistical processing involves advanced analysis of data. A company that has a major rush in their store on Saturday, might do a processing statistical tools and determine the effect of each variables. The results such processing can used to determining the cause of the rush. This is advanced and involves application of statistical methods to derive meaningful conclusions.
