Inserting cases and variables in SPSS

Inserting cases/ variables in SPSS is similar to how data is entered in excel. Go to last row containing cases/variables and enter the new case or variable in the next row. But if the user wants to insert a case or a variable in between the cases or variables, use the following methods

A new case can be inserted in SPSS in multiple ways

  1. Click Edit > Insert Cases; or
  2. Right-click on a row and select Insert Cases from the menu; or
  3. Click the Insert Cases icon

A new variable can also be inserted in multiple ways

  1. Click Edit > Insert Variables
  2. Right-click on a column and select Insert Variables from the menu or
  3. Click the Insert Variables icon

Selecting Cases in SPSS

The DataSet that is being used will have hundreds of cases. But the user might need only few of the cases for analysis. In case when the user might need only a subset of all the data collected for analysis, the Select Cases option can be used. For example, of the data file that was created, the researcher might be interested in analysis of only female students. To select cases in SPSS, perform the following tasks
Data> Select cases.

As mentioned earlier, if only female candidates need to be included in the analysis, in the Select Cases menu, in the Select group, use the second condition, If condition is satisfied.

After selecting If... a new window would pop out which appears as below(Fig 18).The appropriate variable has to be moved to the window using and the necessary condition given. If the data has to selected based on female, the option is Gender=2. Kindly recollect that gender was coded as Male=1, and Female=2. Thus on giving the option, Gender=2, only female candidates will be selected.

Use the number pad given in the window or use the keyboard to enter the condition i.e., (=2).

Click Continue and OK to close the respective windows and return to the data view. Upon completing the task the data view will appear as below. Kindly note that the cases which were excluded will have a diagonal strike through and the included cases will appear without the strike through. SPSS assigns a filter to select the required cases which is displayed in the last column. In this example, kindly note that case numbers 1, 7, 8 and 10 only are selected for the analysis, as the user set female cases to be selected as the condition.
