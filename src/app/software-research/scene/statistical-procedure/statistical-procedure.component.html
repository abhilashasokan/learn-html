Basic Steps for Performing any Statistical Procedure

Creating/Opening a file

  1. A new data file in SPSS can be created as follows, File > New > Data.

  2. If the user wants to open an already existing file File > Open > Data and select the required file.

  3. If the data is in excel, it can opened in the following way, File > Open > Data
    In the Open Data Window, in the Files of type: dropdown list select Excel and click Open.

  4. If the data was opened in SPSS previously in your system it could be quickly opened by File > Recently Used Data

Creating/Opening a file

The active DataSet can be saved in SPSS by File > Save. The DataSet can be saved in different formats. By default SPSS files are saved with .sav extension. Output generated in SPSS can also be saved. By default SPSS saves output with .spv extension.

Closing a file and Exiting SPSS

Exiting SPSS, is similar to any other software, by clicking the close button. As mentioned earlier, when data view and output view are open, there will be two pop-up windows asking for conformation to exit.

The first popup windows enquires whether the user wants to exit. The message displayed is Closing the last Data editor will exit SPSS Statistics. Do you want to proceed?. Click Yes to exit Second popup asking if the user wants to save the output file if any output has been generated. If the dataset has been modified by the user, a pop up window will be displayed enquiring if the changes to the dataset should be saved.
