Introduction

After defining the variables and entering the data, the DataSet is ready for basic transformation and analysis. The data that is entered in SPSS might not be suitable for direct interpretation by the user. It has to be transformed before any analysis can be done.

Data Transformation

Data transformation is the process of converting data from one format or characteristic into another format or characteristics depending on the requirement. Using SPSS, the following data transformations can be performed.

  1. Computing New Variables
  2. Recoding Variables
  3. Automatic Recode
  4. Visual Binning

Computing New Variables

As mentioned above, direct interpretation of data might not yield desired results. For example, the marks scored by 10 students for 5 subjects might not reveal any important pattern or information. In this regard Compute is the most commonly used command used to obtain composite or combined score for variables. This can be performed on the whole data in the files or a selection of it, according to the requirement. SPSS has more than 70 built-in functions to compute new variables based on the users requirement.

These include:

  • Arithmetic functions
  • Statistical functions
  • Distribution functions
  • Logical functions
  • Date and time functions
  • Missing-value functions
  • Cross-case functions
  • String functions

Using the DataSet that was prepared in the previous module(DataSet1.sav), on examination results, a new variable can be computed. For instance, to get a total of all the marks, a new variable Total_marks can be created.

Total _ marks = Accounting+Costing+Income_ Tax+Marketing+Statistics

Steps to compute a new variable

The new variable that was defined Total_marks will be created and it will be listed in the Variable View. The Data View will have the sum total of all the subjects as was defined in the Compute command.

Compute a new variable based on certain conditions

In some instances, it may be required to compute a new variable based on certain conditions. For example, the user may wish to find the total marks of all students who study for more than 10 hours a week. To make this clear, using DataSet1.sav, a new variable is added: study _ hours which is defined as the number of hours a student spends in studying. The new variable is defined in the Variable View and the hours of study of each student is given.
