Recode variables

Recoding is a method by which data of an already existing variable is re-coded or modified so that it can be used for analysis. There are two methods of recoding variables

  1. Recode into same variables
  2. Recode into different variables

a) Recode into same variables

Recode into same variables is the method, by which the changes in data by recoding, is made to the original variable itself. A new variable will not be created. In the example given, for gender, the value labels used are

1= Male, and 2= Female.

If the user wishes to change this value labels as 1= Female and 2= Male, Recode into same variables can be used. The following transformation is required.

The steps for recoding into the same variable are as follows
  1. Open DataSet1.sav
  2. Select Transform
  3. Click on Recode Into Same Variables to open the Recode into Same Variables dialogue box. Please note, by selecting Recode Into Same Variables, the original data will be overwritten.
  4. In the Variable List box, select the variable you wish to recode (e.g., .Gender)
  5. Click on the Old and New Values command button to open the Recode into Same Variables: Old and New Values dialogue box.
  6. In the Old value box, type the old value (i.e, 1)
  7. In the New Value box,type the new value (i.e, 2)
  8. Click on Add
  9. In the Old Value box, type the second old value(i.e, 2)
  10. In the New Value box, type the new second value(i.e, 1)
  11. Click to Add.
  12. In the New Value box, type the new second value(i.e, 1)
  13. Click to Continue and then 0K The output is given below in Fig 9. Compare between the original data and the recoded data.
