Electronic Data Processing

It is simply, the processing of data using electronic devices. Electronic data processing (EDP) is concerned with abstracting meaningful information from large volumes of data. Electronic Data Processing can refer to the use of automated methods to process commercial data Typically, this uses relatively simple, repetitive activities to process large volumes of similar information. Business data processing is a major application of EDP. In business data processing huge quantity of data forms input and the processing results in collapsing the data into small quantity of meaningful information to the users.

Steps in Electronic Data Processing Data processing refers to the process of converting data from one format to another. It transforms plain data into valuable information. The basic steps in data processing are;

  1. Identifying the Data:

    After the information needs are determined, the data for generating that information needs to be identified. The first step in dat processing, is to locate data sources for the information

  2. Designing forms:

    Forms for entering data are designed and linked with databases to enter data into the system.

  3. Data entry:

    The data inputs are keyed into databases through forms.

  4. Data Validation:

    Data validation rules and procedures should be built into the database to avoid data redundancy and errors

  5. Data Storage:

    It is the process of organizing and storing the data in central database by ensuring security to facilitate easy storage and retrieval of data.

  6. Data processing:

    Data is to be processed to get relevant information. It is performed to generate information depend on the nature of data and the information required

  7. Report generation and Communication:

    MIS applications should be developed to generate reports for meeting the information requirements of managers at different levels. The information generated after data processing should be meaningfully presented to the users. The reports may be presented in the form of charts, graphs, presentations etc depending on the format required by the users.
