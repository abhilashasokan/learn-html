Data

Data is used in all forms of human, organizational activity. E.g., in business field (e.g., sales data, revenue, profits, stock price), governance (e.g., unemployment rates, literacy rates) etc. The word data was first used to mean "transmissible and storable computer information" in 1946. Data is the plural of Datum, which means a single fact. Data are raw facts, observations or numbers about entities that need to processed. Data is something simple and seemingly random and useless until it is organized. Data may be in different forms like text, graphics, images, video, sound, numbers etc.

Data can be defined as "a representation of facts, concepts or instructions in a formalized manner, which should be suitable for communication, interpretation or processing by human or electronic machine".

Data is in raw form are plain facts and is not usable as such. Processing of data is required to convert them into usable form. When data are processed, organized, structured or presented in a given context so as to make them useful, they are called Information.

Data collection involves the use of Information Technology (IT) comprising: computers and telecommunications networks (E-Mail, Voice Mail, Internet, telephone, etc.) Computers are important for more quantitative, than qualitative, data collection, storage and retrieval.

Sources of data

The data sources may be generally categorized as:

  1. Internal
  2. External

Internal data source includes records like file records, documents. reports etc. Internal data arises from the routine data processing activity in the organization and it summarizes the organizations operations.

The other source of data is the external environment. It may come from trade publications, government statistics, surveys etc.

Data life cycle

Data life cycle is a sequence of stages that a particular unit of data goes through, from its initial generation or capture to end of its useful life. It describes various stages in the successful management and preservation of data for use and reuse. .

The various stages in data cycle are
  1. Data generation or Capture: Data enters the system, mainly through data entry, or from external sources
  2. Data Integration: Data from different sources are combined to form one homogenous set of data that can be readily analyzed.
  3. Data Analysis: Data used to support organizations objectives and operations
  4. Data Reporting: Present the analyzed data in the form of a report for meaningful presentation.
  5. Data Archiving: Data is removed from all active usage environments. It is no longer processed, used or published but is stored, for future retrieval.
  6. Data Purging: Every copy of data already archived will be deleted, to prevent misuse and overloading.
