Visual Binning Cutpoints with equal percentile

A percentile is a measure indicating the value below which a given percentage of observations falls. 20th percentile refers to the value below which 20% of the observations lie. The user can divide the observations, such that each bin will have an equal number of units. Quartiles, Deciles, Percentiles all work in the same manner. In this example we are going to use deciles. Deciles divide the data into ten equal parts.

The steps given above for visual binning, are the same, except step no. 6, which is explained below

Step 6. In the Make Cutpoints dialogue box, the second radio button of Equal Percentages Based on Scanned Cases is chosen. As decile are used, type 10 in the Width(%):. Number of Cutpoints will be 9. Repeat the remaining steps.

Output appears in the Output Viewer will be as follows. Kindly note that the size of each bin is different but the number of items in each bin are the same.

If the variable chosen contains a relatively small number of distinct values or a large number of cases with the same value, while performing visual binning the user get fewer bins than requested. If there are multiple identical values at a cutpoint, the values will be placed in the same interval; so the actual percentages may not always be exactly equal.

Rank cases

Ranking is the method of recoding the data into ranks. Rank cases is very commonly used option of SPSS. It is used to order the data from smallest to largest or largest to smallest.

Using the DataSet1 example, in order to create a rank of test scores, perform the following steps.

  1. Transform > Rank cases
  2. In the rank cases dialogue box, move the required variable (Marks of Accounting) into the Variable(s): box.
  3. In the option Assign Rank 1 to, select the largest value radio button as the ranking is done on marks scored.

  4. Click 0K. A new variable will be created with the ranks. Check the Data View and the Variable View
