Automatic recode

Automatic recode helps in converting a string variable into a numerically coded variable, without manually doing it for each item. This procedure assigns each unique category a numeric code, then saves the converted values as a new variable.

For example, if Gender was entered into the DataSet as string variable viz. 'Male' and 'Female' , Automatic recoding helps to code the data into 1= Male and 2= Female or 1=Female or 2= Male.

To illustrate, Using the DataSet1.sav, Create a new variable as Gender_String, and select the Type as String in the Variable View.

In the Data View for the variable Gender_String, enter Male or Female for each case.

Steps for Automatic Recode

  1. Select Transform> Automatic recode
  2. Automatic Recode dialogue box will appear.
  3. In the Automatic Recode option move the variable (Gender_string), into the box.
  4. In the option New Name, enter a name (Gender _ Autorecoded) for a new variable. The automatically recoded values will be displayed under this new name.
  5. Click Add New Name
  6. In the Recode Starting from, there are two options. This option decides the values to be automatically assigned to the string variables. Values (1, 2, 3... ) are assigned alphabetically or reverse alphabetically depending on the variable names.
  7. Lowest value: Recoding of String Variables Alphabetically, (A-> Z), Female=l and Male=2.
  8. Highest value: Recoding of String Variables Reverse Alphabetically, (Z->A) Male=l and Female=2. In this example, if the user wants to alphabetically recode the string variables. i.e., Female=l and Male=2. In the Recode Starting from select the option Lowest value.
  9. Click 0K
The output will be displayed as follows and Female has been coded as 1 and Male as 2.
